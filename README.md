# MERCADO LIBRE
Test Práctico - Frontend

## Instalación

Requiere [Node.js](https://nodejs.org/) v14+ to run.
Requiere [Yarn](https://yarnpkg.com/) v1.22+.

Instalar dependencias

```sh
git clone https://github.com/josefes23/meli.git
cd meli
npm install -g yarn
yarn install
```

## Comandos

PROD:

```sh
yarn start
```

DEV:

```sh
yarn dev
```

LINTER:
```sh
yarn lint:check
```
